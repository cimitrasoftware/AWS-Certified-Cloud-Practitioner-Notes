**Dedicated Hosts**
- Reserve an entire physical server for exclusive use by your entity/company
- If the company wants, this can grant them full control of the machine
- This is helpful for security or licensing compliance necessities

**Dedicated Instances**
- Run Instances on hardware dedicated to the entity/company
- Where the Dedicated Instances are run IS NOT in the company's control
- No other instances will run on hardware that is running the company's Dedicated Instances
